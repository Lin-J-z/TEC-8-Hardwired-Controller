## 二、项目概述

本设计为 TEC-8 实验箱实现了一个**硬布线控制器**，取代原有的微程序控制器。控制器采用组合逻辑实现指令译码，根据指令码（IR[7:4]）和时序节拍（W1/W2/W3）生成全部微操作控制信号。

**支持的指令**（11条）：
- 算术逻辑：ADD、SUB、AND、INC、NOT、CMP
- 数据传送：LD、ST、MOV
- 转移：JC、JZ、JMP
- 输出：OUT
- 停机：STP

---

## 三、运行模式

| 模式 | DP 开关 | 行为 |
|------|---------|------|
| 单步 | 向上 | 每按一次 QD 执行一个节拍（W1/W2/W3） |
| 连续 | 向下 | 按一次 QD 连续运行至 STP 指令停机 |

**控制台操作模式**：

| SWC/SWB/SWA | 功能 |
|-------------|------|
| `000` | 取指（程序运行） |
| `001` | 写存储器 |
| `010` | 读存储器 |
| `011` | 读寄存器 |
| `100` | 写寄存器 |

## 五、ADD 指令验证（简要）

**预置**：
- 地址 0x00 写入 `00010001`（ADD R0, R1）
- R0 = 0x05，R1 = 0x03

**单步执行**（模式 000，DP=向上）：
- W1：LIR=1，PCINC=1
- W2：S3=1, S0=1, CIN=1, ABUS=1, DRW=1, LDZ=1, LDC=1
- W3：全部熄灭

**结果**：R0 = 0x08 ?

**连续运行**（DP=向下）：
- 地址 0x01 写入 `11100000`（STP）
- 按一次 QD 启动，遇 STP 停机，STOP 灯常亮

## 七、烧录流程

1. Quartus II 新建工程，器件选 `EPM7128SLC84-15`
2. 添加 `src/basic_task.v` 到工程
3. 导入 `src/basic_task.qsf` 引脚分配
4. 全编译（Processing → Start Compilation）
5. 连接 USB-Blaster，烧录 `.pof` 文件
6. TEC-8 控制器开关拨到**朝上**（硬布线）
